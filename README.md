## Getting Started

### Requirements

1. Windows 7 or higher

### Installation

1. Extract file ```hornbill-keyboard-tester-master.zip```.
2. Rename folder ```hornbill-keyboard-tester-master``` to ```hornbill-keyboard-tester```.

### Usage

1. Double click file ```electron.exe``` to start app.
